##### Yêu cầu (Cài đặt NodeJs cho máy tính)

##### Cài đặt đồ án
1. Cài đặt thư viện: (thự hiện cầu lệnh bên dưới trong terminal)
```
npm i
```
2. Chạy Server:

```
npm start
```
